SimpleQuery is a collection of scripting languages developed in Python revolving around a simple space-separated syntax concept.
However its base is a interpreted, high level and general purpose scripting language. SimpleQuery only works for Python 3.10 and later.
Download the entire SimpleQuery repository and run console.py in order to work with it. Yep, it's that simple!
